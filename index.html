HTML5 + CSS3

HTML5

Es un lenguaje de maquetación, crea la estructura de un sitio web.

Se utiliza para la estructura de una web, el contenido, texto, imágenes, videos, etc.

CSS

Es un lenguaje que maneja cascada de estilos.

CSS es la estética, los colores, dónde de ubica cada contenido.

Ejemplo: Construyendo una casa

HTML serían las vigas de una casa.

CSS Es la pintura y la decoración.
